# flask_bp_tutorial

Simple flask app that uses blueprints

### Installation

```git clone https://github.com/glnnlhmn/flask_bp_tutorial.git```

```cd flask_bp_tutorial```

```python -m venv venv```

| Windows | Linux |
| -------- | --------- |
| ```venv/source/activate``` | ```source venv/bin/activate```  |

```pip install -r requirement.txt```
